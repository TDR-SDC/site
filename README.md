## **Prerequisites**
- NodeJS
- npm

## **Build and go live**
```bash
git clone https://github.com/krush/defianz_site
cd defianz_site
npm install
npm start
```
Go to `localhost:3000` or `127.0.0.1:3000` to see the website live
